Stable tag: 1.0.0
License: Apache 2.0

== Description ==

Bulgarisation Notifications Fix changes notifications after order placement to include
the transaction ID in the NRA receipt.

== Instalation ==

Download latest release of the plugin.

=== Install from within WordPress ===

1. Visit the plugins page within your dashboard and select ‘Add New’;
2. Choose Upload and upload the archive;
3. Activate Bulgarisation Notifications Fix from your Plugins page;

=== Install manually ===

1. Upload the ‘bulgarisation-notifications-fix’ folder to the /wp-content/plugins/ directory;
2. Visit the plugins page within your dashboard.
3. Activate the Bulgarisation Notifications Fix plugin through the ‘Plugins’ menu in WordPress;
